隱私權政策

生效日期：［請填寫日期］

UrgeCare App 重視您的隱私。我們不會蒐集、儲存或傳送任何個人資料到遠端伺服器。所有資料僅保留在您的裝置上。

1. 資料蒐集

本應用程式不會蒐集任何可識別您身分的個人資料（PII）。

2. 本地儲存

所有資料（如日記、待辦事項、番茄鐘等）皆儲存在瀏覽器的本地資料庫（IndexedDB），不會上傳或同步到外部。

3. 分析與追蹤

本應用程式不使用任何 cookie、不進行使用者行為分析，也不含任何追蹤程式碼。

4. 第三方服務

我們不使用任何會存取您資料的第三方 API 或 SDK。

5. 資料控制

您可透過瀏覽器設定手動刪除所有本地資料。若您移除此應用或清除快取，資料將一併被刪除，且無法復原。

6. 聯絡方式

若您對本政策有任何疑問，歡迎透過 GitHub 專案頁聯絡我們： https://github.com/ang2289/urgecare-app
