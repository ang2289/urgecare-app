這是我們一起做的 UrgeCare（戒習助手）App 專案，使用 React + Vite + TypeScript + Capacitor（Android）。

目前已完成的功能與檔案：
- App.tsx：分頁切換（SOS 預設首頁、Diary、Todos、Settings、SupportWall）
- SOS.tsx：延遲倒數計時，結束時觸發震動＋通知，並自動寫入日記＋待辦（含 10 分鐘去重），有替代行為列表。
- Diary.tsx：日記功能（新增、搜尋、語音輸入、編輯、刪除、CSV 匯出含 BOM），語音避免重複輸入。
- Todos.tsx：待辦功能（新增、完成勾選、刪除、清空），支援 storage.ts 本地儲存，跨頁切換後資料不會消失。
- Settings.tsx：可設定 SOS 預設分鐘數、冷卻分鐘數。
- SupportWall.tsx：支持牆，可上傳照片（已改用 Capacitor Filesystem 儲存），顯示家人或伴侶照片。
- storage.ts：統一封裝日記、待辦、設定的 localStorage，包含 addEntrySmart / addTodoSmart（冷卻去重）。
- reset.css / theme.css：大字體、高對比、卡片風格、圓角、間距一致。
- index.html / main.tsx 已設定好 Vite + React 啟動。
- 已修正 StrictMode 導致的 Todos 無限更新迴圈。

注意事項：
- CSV 匯出在 Excel 打開不會亂碼（UTF-8 BOM 已加）。
- Android 原生需要權限：通知權限、檔案存取權限。
- 開發測試時 favicon.ico 需放在 public/ 避免 404。
